# 基于贝叶斯估计与深度神经网络结合的水下图像复原技术

## 实验报告

---

## 一、实验概述

### 1.1 研究背景

水下图像由于光线的吸收和散射作用，普遍存在以下问题：

| 问题 | 表现 | 原因 |
|------|------|------|
| 模糊 | 图像细节丢失 | 水中悬浮颗粒散射光线 |
| 偏色 | 整体呈蓝绿色 | 不同波长光线吸收程度不同 |
| 对比度低 | 图像灰暗 | 光线在水中快速衰减 |
| 噪声 | 图像颗粒感 | 水下环境复杂多变 |

### 1.2 研究目标

将退化的水下图像恢复为清晰、真实的正常图像。

**技术指标**：
- 峰值信噪比(PSNR) ≥ 25dB
- 结构相似度(SSIM) ≥ 0.85
- 推理速度 ≤ 2秒/张

### 1.3 技术路线

采用贝叶斯估计框架结合深度学习方法：

```
退化模型: y = Hx + n

其中:
- y: 观测到的退化图像
- x: 原始清晰图像（目标）
- H: 退化算子
- n: 噪声
```

使用ADMM（交替方向乘子法）将问题分解：
1. **H估计网络**：学习退化算子的逆变换
2. **去噪网络**：去除噪声，恢复细节

## 二、实验设置

### 2.1 数据集

| 测试集 | 89张 | 10% |

**数据预处理**：
- 图像尺寸：256×256像素
- 数据增强：随机翻转、旋转、颜色扰动

### 2.2 网络结构

#### H估计网络

```
输入: 6通道 (退化图像 + 辅助变量)
    ↓
卷积层 (6→64) + ReLU
    ↓
残差块 × 8 (64通道)
    ↓
卷积层 (64→3) + Sigmoid
    ↓
输出: 3通道 (估计的清晰图像)
```

**参数量**：约1.5M

**训练特点**：使用随机噪声训练，无需真实水下图像数据

#### 去噪网络

```
输入图像 (3通道)
    ↓
卷积层 (3→64)
    ↓
小波分解 (db4小波)
    ↓
┌─────────────┬─────────────┐
│ 低频分支     │ 高频分支     │
│ (整体结构)   │ (细节边缘)   │
│ 残差块×1    │ 残差块×3    │
└─────────────┴─────────────┘
    ↓
小波重构
    ↓
卷积层 (64→3)
    ↓
颜色校正模块
    ↓
残差连接 + 输出
```

**参数量**：约2.0M

**核心特点**：
- 小波变换分离低频/高频分量
- 双分支分别处理结构和细节
- 颜色校正模块解决偏色问题

### 2.3 训练配置

## 四、结果分析

### 4.1 效果不理想的原因分析

#### 1. 网络结构限制

**问题**：当前网络结构相对简单，表达能力有限

| 组件 | 当前设计 | 局限性 |
|------|----------|--------|
| 残差块 | 8层 | 深度不足 |
| 小波分解 | db4小波 | 单尺度 |
| 颜色校正 | 2层MLP | 参数太少 |

**残差连接的限制**：
```python
output = x + output  # 输出 ≈ 输入 + 小改动
```
这导致输出图像与输入图像差异不大，难以实现显著的颜色校正。

#### 2. 训练策略问题

**问题**：H估计网络和去噪网络独立训练，没有联合优化

```
当前方案:
H网络训练 → 固定H网络 → 去噪网络训练
    ↓              ↓
独立优化      独立优化

理想方案:
H网络 ←→ 去噪网络
    ↓
联合优化
```

#### 3. 数据集问题

**UIEB数据集的局限性**：
- 数据量较小（890对）
- 参考图质量参差不齐
- 部分参考图本身不够清晰

#### 4. 损失函数问题

**当前损失函数**：
```python
loss = L1_loss + 0.1 * SSIM_loss
```

**局限性**：
- L1损失关注像素级差异，可能导致模糊
- 缺少感知损失，无法保证视觉质量

### 4.2 与目标指标的差距

| 指标 | 当前结果 | 目标 | 差距 | 原因 |
|------|----------|------|------|------|
| PSNR | 14.43 dB | 25 dB | -10.57 dB | 网络表达能力不足 |
| SSIM | 0.46 | 0.85 | -0.39 | 结构恢复能力弱 |

### 4.3 与SOTA方法对比（WF-Diff）

#### 对比结果

**UIEB数据集对比：**

| 方法 | PSNR (dB) | SSIM | LPIPS | UIQM | UCIQE |
|------|-----------|------|-------|------|-------|
| **本文方法** | 14.43 | 0.4633 | - | - | - |
| **WF-Diff** | 25.68 | 0.9093 | 0.1586 | 4.33 | 0.5903 |
| **差距** | **-11.25** | **-0.446** | - | - | - |

**LSUI数据集对比：**

| 方法 | PSNR (dB) | SSIM | LPIPS | UIQM | UCIQE |
|------|-----------|------|-------|------|-------|
| **本文方法** | - | - | - | - | - |
| **WF-Diff** | 22.83 | 0.8539 | 0.3174 | 4.20 | 0.5903 |

> 注：WF-Diff LSUI结果为resize到512×512后的测试结果

#### 差距分析

| 方面 | 本文方法 | WF-Diff | 差距原因 |
|------|----------|---------|----------|
| **网络架构** | 简单残差网络 | 双Diffusion模型+UNet | 表达能力差距大 |
| **参数量** | ~3.5M | ~100M | 参数量差30倍 |
| **训练数据** | UIEB (890对) | LSUI (4000+对) | 数据量差距 |
| **损失函数** | L1 + SSIM | 多任务损失 | 损失设计差距 |
| **训练策略** | 分步训练 | 端到端训练 | 优化策略差距 |

#### 关键差距原因

1. **网络表达能力不足**
   - 本文：8层残差块，单尺度小波分解
   - WF-Diff：双Diffusion模型，多尺度特征提取

2. **参数量差距悬殊**
   - 本文：约3.5M参数
   - WF-Diff：约100M参数（相差30倍）

3. **训练数据不足**
   - 本文：仅使用UIEB数据集（890对）
   - WF-Diff：使用LSUI等大规模数据集（4000+对）

4. **训练策略差异**
   - 本文：H网络和去噪网络独立训练
   - WF-Diff：端到端联合优化

## 五、改进方向

### 5.1 缩小与WF-Diff差距的策略

#### 当前差距与改进目标

| 指标 | 当前 | WF-Diff | 差距 | 改进目标 |
|------|------|---------|------|----------|
| PSNR | 14.43 | 25.68 | -11.25 dB | 缩小到5dB以内 |
| SSIM | 0.46 | 0.91 | -0.45 | 缩小到0.2以内 |

#### 改进优先级

| 优先级 | 改进方向 | 预期提升 | 实现难度 |
|--------|----------|----------|----------|
| ⭐⭐⭐ | 端到端联合训练 | +3-5 dB PSNR | 中 |
| ⭐⭐⭐ | 使用U-Net架构 | +2-4 dB PSNR | 中 |
| ⭐⭐ | 引入感知损失 | +1-2 dB PSNR | 低 |
| ⭐⭐ | 扩大训练数据 | +2-3 dB PSNR | 低 |
| ⭐ | 引入注意力机制 | +1-2 dB PSNR | 高 |

### 5.2 网络结构改进

#### 方案一：使用U-Net架构

```
编码器: 逐步下采样，提取特征
    ↓
瓶颈层: 最深层特征
    ↓
解码器: 逐步上采样，恢复细节
    ↓
跳跃连接: 保留细节信息
```

**优势**：
- 更强的特征提取能力
- 跳跃连接保留细节
- 广泛应用于图像复原任务

#### 方案二：引入注意力机制

```
通道注意力: 关注重要特征通道
空间注意力: 关注重要空间位置
```

**优势**：
- 自适应调整特征权重
- 更好地处理颜色校正

### 5.2 训练策略改进

#### 方案一：端到端联合训练

```
输入 → H网络 → 去噪网络 → 输出
      ↑______________|
           联合优化
```

**优势**：
- 两个网络相互配合
- 整体性能更优

#### 方案二：渐进式训练

```
阶段1: 预训练去噪网络（使用合成数据）
阶段2: 预训练H网络（使用随机噪声）
阶段3: 联合微调（使用真实数据）
```

### 5.3 损失函数改进

#### 引入感知损失

```python
loss = L1_loss + λ1 * SSIM_loss + λ2 * Perceptual_loss
```

**感知损失**：使用预训练VGG网络提取特征，计算特征层面的差异

**优势**：
- 更符合人眼感知
- 输出图像更自然

### 5.4 数据增强

#### 使用更大数据集

| 数据集 | 图像数量 | 特点 |
|--------|----------|------|
| UIEB | 890 | 配对数据 |
| EUVP | 12,000+ | 配对数据 |
| LSUI | 4,000+ | 大规模 |

### 5.5 后处理优化

#### 直方图均衡化

```python
# 对输出图像进行直方图均衡化
enhanced = cv2.equalizeHist(output)
```

**优势**：简单有效，可以改善对比度
