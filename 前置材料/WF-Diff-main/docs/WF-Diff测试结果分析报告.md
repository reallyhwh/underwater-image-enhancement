## 四、结果差异分析

### 4.1 主要差异原因

| 原因 | 影响程度 | 详细说明 |
|------|----------|----------|
| **Resize损失** | 高 | LSUI图像从1280×1024缩放到512×512，信息损失约75% |
| **数据集难度** | 中 | LSUI包含更多样化的水下场景和退化类型 |
| **尺寸不匹配** | 低 | GT与输出尺寸不完全匹配时的插值误差 |

### 4.2 Resize对指标的影响估算

根据经验，resize操作对指标的影响约为：

- PSNR下降：1-3 dB
- SSIM下降：0.02-0.08
- LPIPS上升：0.05-0.20

LSUI的实际下降幅度在合理范围内。

### 4.3 数据集特性差异

| 特性 | UIEB | LSUI |
|------|------|------|
| 图像来源 | 单一数据源 | 多数据源合成 |
| 退化类型 | 相对单一 | 多种退化 |
| 参考图像 | 真实清晰图像 | 合成参考图像 |
| 场景多样性 | 较低 | 较高 |

## 五、结论与建议

### 5.1 测试结论

1. **WF-Diff在UIEB数据集上表现优秀**：PSNR 25.68 dB，SSIM 0.9093，达到SOTA水平
2. **LSUI结果受限于显存**：由于需要resize，指标有所下降，但仍可作为对比基准
3. **UCIQE指标稳定**：两个数据集的UCIQE相同，说明色彩恢复能力稳定

### 5.2 使用建议

| 场景 | 建议使用的指标 |
|------|----------------|
| 论文对比 | UIEB结果更准确，优先使用 |
| 方法验证 | 两个数据集结果都可作为参考 |
| 性能分析 | 需注明LSUI使用了resize策略 |

### 5.3 后续工作建议

1. **获取更准确的LSUI基准**：使用更大显存的GPU（如12GB+）运行原始尺寸测试
2. **补充其他数据集**：如EUVP、U45等，增加对比的全面性
3. **记录resize影响**：在论文中说明测试条件和限制

### 6.2 指标说明

| 指标 | 全称 | 范围 | 意义 |
|------|------|------|------|
| PSNR | Peak Signal-to-Noise Ratio | 通常15-40 dB | 像素级重建质量 |
| SSIM | Structural Similarity Index | [0, 1] | 结构相似性 |
| LPIPS | Learned Perceptual Image Patch Similarity | [0, 1] | 感知相似度 |
| UIQM | Underwater Image Quality Measure | 通常0-10 | 水下图像综合质量 |
| UCIQE | Underwater Colour Image Quality Evaluation | 通常0-1 | 水下图像色彩质量 |
