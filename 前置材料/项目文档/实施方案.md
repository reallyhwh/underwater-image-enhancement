# 基于贝叶斯估计与深度神经网络结合的水下图像复原技术 - 实施方案

---

## 一、课题目标拆解

### 1.1 核心目标
将退化水下图像（模糊、偏色、细节丢失）恢复为清晰、真实的正常图像

### 1.2 技术指标
- 峰值信噪比(PSNR) ≥ 25dB
- 结构相似度(SSIM) ≥ 0.85
- 推理速度 ≤ 2秒/张（GPU环境）

---

## 二、总体技术路线

```
退化图像 → 贝叶斯建模 → ADMM问题分解 → 双网络迭代求解 → 清晰图像
                     ↓
            ┌────────┴────────┐
            ↓                 ↓
    H估计子网络       小波去噪子网络
    (无需真实数据)    (需水下数据集)
```

## 三、具体实施步骤

### 第一阶段：环境搭建与数据准备（1-2周）

| 任务 | 内容 | 工具/资源 |
|------|------|-----------|
| 环境配置 | Python 3.8+、PyTorch 1.10+、CUDA 11.x | Anaconda |
| 依赖安装 | OpenCV、Pillow、numpy、scipy、pywt | pip/conda |
| 数据集获取 | UIEB、EUVP、LSUI数据集 | 公开数据集下载 |
| 数据预处理 | 图像裁剪(256×256)、归一化、数据增强 | Python/OpenCV |

**数据集说明**：
- **UIEB**：包含890张水下图像及对应清晰图像
- **EUVP**：成对数据集，包含水下/清晰图像对
- **合成数据**：用清晰图+水下退化模型合成训练数据

---

### 第二阶段：退化矩阵H估计子网络训练（2-3周）

#### 2.1 网络设计

```
输入: 随机高斯噪声 (batch, 3, H, W)
  ↓
卷积层(3→64) → Relu
  ↓
残差块 × 8 (64通道)
  ↓
卷积层(64→3) → Sigmoid
  ↓
输出: 估计的退化矩阵逆变换
```

#### 2.2 训练策略
- **无数据训练**：不需要真实水下图像，用随机噪声训练
- **损失函数**：MSE Loss
- **优化器**：Adam (lr=1e-4)
- **Epochs**：100
- **Batch Size**：8

#### 2.3 核心思路
网络学习的是「退化算子的逆变换」，不需要成对数据集，用随机噪声就能完成训练

---

### 第三阶段：小波去噪子网络训练（2-3周）

#### 3.1 网络架构

```
输入: 退化图像
  ↓
小波分解 (Haar/wave-db4)
  ↓
├── 低频分支：卷积 → 残差块 → 卷积
├── 高频分支：卷积 → 残差块 × 3 → 卷积
  ↓
小波重构
  ↓
输出: 去噪图像
```

#### 3.2 训练配置
- **数据集**：UIEB/EUVP成对数据
- **损失函数**：L1 Loss + SSIM Loss（加权组合）
- **数据增强**：随机翻转、旋转、亮度调整
- **优化器**：Adam (lr=1e-4)
- **Epochs**：150
- **Batch Size**：4

#### 3.3 先验知识
网络学习「正常水下图像应该长什么样」，掌握水下图像的颜色分布、纹理特征

---

### 第四阶段：ADMM迭代框架搭建（2周）

#### 4.1 核心迭代公式

```python
# 初始化
x⁰ = y (退化图像), v⁰ = 0, u⁰ = 0

# 迭代过程 (k = 0, 1, 2, ...)
xᵏ⁺¹ = argmin ||y - Hx||² + λ||Wx||₂² + (ρ/2)||x - vᵏ + uᵏ||²
       → 用H估计子网络求解

vᵏ⁺¹ = argmin λ||Wx||₂² + (ρ/2)||xᵏ⁺¹ - v + uᵏ||²  
       → 用小波去噪子网络求解

uᵏ⁺¹ = uᵏ + xᵏ⁺¹ - vᵏ⁺¹
```

#### 4.2 超参数设置

| 参数 | 推荐值 | 作用 |
|------|--------|------|
| ρ (penalty) | 0.01 | ADMM惩罚因子 |
| λ (regularization) | 0.001 | 正则化系数 |
| max_iter | 20 | 最大迭代次数 |
| tol | 1e-4 | 收敛阈值 |

#### 4.3 停止条件
- 达到最大迭代次数 `max_iter`
- 连续两次迭代的图像差异 < `tol`

## 六、关键难点与解决方案

| 难点 | 解决方案 |
|------|----------|
| H估计网络训练不稳定 | 使用谱归一化、调整学习率 |
| 去噪网络过拟合 | 增加数据增强、使用Dropout |
| ADMM收敛慢 | 调整ρ参数、使用自适应步长 |
| 颜色失真 | 在去噪网络中加入色彩校正分支 |
