### 【阶段三】方法设计

#### 3.1 技术方案设计

```
技术路线：
┌─────────────────────────────────────────────────┐
│  退化图像 y                                     │
│      ↓                                          │
│  贝叶斯建模                                      │
│      ↓                                          │
│  ADMM问题分解                                    │
│      ↓                                          │
│  ┌───────────┬───────────┐                      │
│  │ H估计网络  │ 去噪网络  │                      │
│  │ (无数据)  │ (需数据)  │                      │
│  └───────────┴───────────┘                      │
│      ↓                                          │
│  迭代求解                                        │
│      ↓                                          │
│  清晰图像 x                                      │
└─────────────────────────────────────────────────┘
```

#### 3.2 网络架构设计

| 网络 | 结构 | 作用 |
|------|------|------|
| **H估计网络** | 残差网络（8个残差块） | 学习退化算子的逆变换 |
| **去噪网络** | 小波分解 + 双分支处理 | 去除噪声，恢复细节 |
| **ADMM框架** | 迭代优化算法 | 组合两个网络求解 |

| 产出物 | 说明 |
|--------|------|

| 超参数设置 | 学习率、迭代次数等 |

#### 5.1 训练流程

```
模型训练：

│   ├── Python 3.10

│   └── 依赖库安装
│

│   ├── 输入：随机噪声
│   ├── 输出：模型权重 best_model.pth
│   └── 时间：约10-20分钟
│
├── 训练去噪网络
│   ├── 输入：成对水下图像
│   ├── 输出：模型权重 best_model.pth
│   └── 时间：约30-60分钟
│
└── 训练监控
    ├── TensorBoard可视化
    └── Loss曲线观察
```

#### 5.2 训练代码结构

```python
# 训练循环
for epoch in range(epochs):
    for batch in dataloader:
        optimizer.zero_grad()      # 清零梯度
        output = model(input)      # 前向传播
        loss = criterion(output, target)  # 计算损失
        loss.backward()            # 反向传播
        optimizer.step()           # 更新参数
```

#### 5.3 参数更新原理

```
参数更新流程：
├── 步骤1: 前向传播
│   └── 输入图像 → 网络计算 → 输出结果
│
├── 步骤2: 计算损失
│   └── 比较输出和目标，计算差距
│
├── 步骤3: 反向传播
│   └── 自动计算每个参数对损失的影响（梯度）
│
└── 步骤4: 更新参数（自动）
    └── weight = weight - lr × gradient
```

```
实验验证：
```

#### 6.2 评估指标说明

| 指标 | 全称 | 说明 | 目标值 |
|------|------|------|--------|
| **PSNR** | Peak Signal-to-Noise Ratio | 峰值信噪比，衡量图像质量 | ≥ 25 dB |
| **SSIM** | Structural Similarity Index | 结构相似度，衡量结构保持 | ≥ 0.85 |
| **UIQM** | Underwater Image Quality Measure | 水下图像质量评价指标 | 越高越好 |

#### 6.3 实验结果示例

| 方法 | PSNR | SSIM | UIQM |
|------|------|------|------|
| 输入图像 | 18.5 | 0.72 | 0.85 |
| CLAHE | 21.2 | 0.78 | 0.92 |
| U-Net | 24.5 | 0.83 | 1.05 |
| **本方法** | **26.8** | **0.87** | **1.15** |

## 六、常见问题与解决方案

### 6.1 环境问题

| 问题 | 解决方案 |
|------|----------|
| PyTorch 是 CPU 版本 | 重新安装 GPU 版本 |
| pip 无写入权限 | 修改目录权限或用 `python -m pip` |
| 依赖版本冲突 | 使用虚拟环境隔离 |

### 6.2 训练问题

| 问题 | 解决方案 |
|------|----------|
| Loss 不下降 | 检查训练逻辑，确保目标正确 |
| 显存不足 | 减小 batch_size |
| 训练太慢 | 使用 GPU，优化数据加载 |

### 6.3 实验问题

| 问题 | 解决方案 |
|------|----------|
| 指标不达标 | 调整网络结构或超参数 |
| 效果不稳定 | 增加训练数据或数据增强 |
| 对比不公平 | 统一实验设置和评估标准 |
