# 使用深度神经网络的无内循环逆问题求解方法

**An Inner-loop Free Solution to Inverse Problems using Deep Neural Networks**

**会议**: NeurIPS 2019

**作者**: Kai Fan, Qi Wei, Lawrence Carin, Katherine Heller (杜克大学)

---

## 摘要

我们提出了一种新方法，使用深度学习技术加速流行的交替方向乘子法（ADMM）求解逆问题。ADMM更新包括近端算子、包含大矩阵求逆的最小二乘回归，以及更新对偶变量的显式解。通常，由于先验的难处理性和矩阵求逆，需要内循环来求解前两个子最小化问题。为了避免这些缺点或限制，我们提出了一种**无内循环**的更新规则，使用两个预训练的深度卷积架构。更具体地说，我们学习了一个条件去噪自编码器，它在第一个子最小化问题中对真实值施加隐式的数据依赖先验/正则化。这种设计遵循经验贝叶斯策略，导致所谓的摊销推断。对于第二个子问题中的矩阵求逆，我们学习了一个卷积神经网络来近似矩阵求逆，即逆映射是通过将输入通过学习的前向网络来学习的。注意，训练这个神经网络不需要真实值或测量值，即数据无关。在合成数据和真实数据集上的大量实验表明，与使用内循环求解逆问题的传统ADMM解相比，所提方法具有更高的效率和准确性。

## 1 引言

大多数逆问题直接表述为与前向模型相关的优化问题。前向模型将未知信号（即真实值）映射到关于它们的获取信息，我们称之为数据或测量值。这种映射或前向问题通常取决于将真实值链接到测量值的物理理论。求解逆问题涉及学习从测量值到真实值的逆映射。具体来说，它从少量退化或噪声测量中恢复信号。这通常是病态的。

最近，深度学习技术已成为优秀的模型，并在模式分析（无监督）、分类（监督）、计算机视觉、图像处理等应用中获得了广泛的成功。利用深度神经网络帮助求解逆问题最近已被探索，基于深度学习的方法在许多具有挑战性的逆问题中取得了最先进的性能，如超分辨率、图像重建、自动着色。

更具体地说，最近的工作中，从测量值**y**到真实值**x**的端到端映射是从训练数据中学习的，然后应用于测试数据。因此，传统逆问题求解器所需的复杂推理方案被替换为通过预训练网络输入新测量值，这更加高效。

在本文中，我们提出了一个无内循环框架，意思是不需要迭代算法来求解子问题，使用逆问题的分裂策略。两个子问题的交替更新是通过两个预训练的深度神经网络进行的，即一个使用基于摊销推断的去噪卷积自编码器网络进行近端操作，另一个使用结构化卷积神经网络进行与前向模型相关的大矩阵求逆。因此，ADMM中每次迭代的计算复杂度与信号的维度成线性关系。

**贡献**: 本文的主要贡献包括：
1. 使用去噪自编码器神经网络学习隐式先验/正则化器，基于摊销推断
2. 使用结构化卷积神经网络学习大矩阵的逆，无需使用训练数据
3. 上述每个网络都可以用于加速现有的ADMM逆问题求解器

## 2 线性逆问题

**符号说明**: 可训练网络用花体字母表示，如𝒜；固定网络用斜体表示，如A。

低维测量值表示为**y** ∈ R^m，它是由高维真实值**x** ∈ R^n通过线性算子A降维得到的，使得**y** = A**x**。注意通常n ≥ m，这使得要估计的参数数量不小于手头的数据点数量。这给在新观测**y**上找到解**x**施加了一个病态问题，因为A是一个欠定测量矩阵。

在本文中，我们关注流行的ADMM框架，因为其低计算复杂度和最近在求解大规模优化问题方面的成功。更具体地说，优化问题表述为：

```
min ½||y - Az||² + λR(x)
s.t. x = z
```

其中引入的辅助变量**z**被约束为等于**x**，R(x)捕获先验/正则化所促进的结构。

增广拉格朗日函数为：

```
L(x, z, u) = ||y - Az||² + λR(x; y) + ⟨u, x - z⟩ + β||x - z||²
```

其中**u**是拉格朗日乘子，β > 0是惩罚参数。

ADMM将最小化分解为两个子问题，分别关于**x**和**z**的最小化。迭代如下：

```
x^(k+1) = argmin_x λR(x; y) + β||x - z^k + u^k||²
z^(k+1) = argmin_z ||y - Az||² + β||x^(k+1) - z + u^k||²
u^(k+1) = u^k + x^(k+1) - z^(k+1)
```

如果先验R选择得当，如||x||₁，(3)有闭式解（软阈值解）。然而，对于一些更复杂的正则化，求解(3)是非平凡的，可能需要迭代方法。求解(4)需要矩阵求逆，通常应用共轭梯度下降（CG）来更新**z**。因此，求解(3)和(4)通常是繁琐的。由于先验和求逆的难处理性，需要内循环来求解这两个子最小化问题，导致较大的计算复杂度。为了避免这些缺点或限制，我们提出了一种使用两个预训练深度卷积架构的**无内循环**更新规则。

## 3 无内循环ADMM

### 3.1 使用条件近端算子的摊销推断求解x

求解子问题(3)等价于找到近端算子PR(v; y)的解。如果我们施加一阶必要条件，有：

```
x = PR(v; y) ⇔ v - x ∈ ∂R(·; y)(x)
```

其中∂R(·; y)是偏导数算子。为简单起见，我们定义另一个算子F = I + ∂R(·; y)。因此，我们的目标是学习这样一个逆算子，它将**v**投影到先验子空间。

在这项工作中，我们提出了一种隐式的基于示例的先验，它没有真正的贝叶斯解释，但有助于模型优化。与此先验一致，我们定义了由θ参数化的隐式近端算子G_θ(x; v, y)来近似未知的F^(-1)。更具体地说，我们提出了一种称为条件像素重排去噪自编码器（cPSDAE）的神经网络架构作为算子G，其中像素重排意味着周期性地重新排列每个通道中的像素，将高分辨率图像映射到尺度为r的低分辨率图像，并将通道数增加到r²。这使我们能够变换**v**使其与**y**具有相同的尺度，并将其与**y**连接作为cPSDAE的输入。

### 3.2 z的无求逆更新

虽然写出关于**z**的子问题(4)的闭式解很简单，但显式计算这个解是非平凡的。在(7)中，A^⊤是矩阵A的转置。如我们所提到的，右边的项K涉及昂贵的矩阵求逆，计算复杂度为O(n³)。在某些特定假设下，例如A是循环矩阵，这个矩阵求逆可以用快速傅里叶变换加速，其复杂度为O(n log n)。通常，基于梯度的更新在每次迭代中具有线性复杂度，因此总体复杂度为O(n_int log n)，其中n_int是迭代次数。

在这项工作中，我们将通过设计神经网络来显式学习这个矩阵求逆。注意K只依赖于A，因此可以预先计算以备将来使用。这个问题可以通过应用Sherman-Morrison-Woodbury公式简化为更小规模的矩阵求逆：

```
K = (βI + A^⊤A)^(-1) = β^(-1)[I - A^⊤(βI + AA^⊤)^(-1)A]
```

因此，我们只需要求解m × m维的矩阵求逆，即估计B。我们提出了一种方法，通过可训练的深度卷积神经网络C_φ ≈ B来近似它，由φ参数化。

### 3.3 cPSDAE的对抗训练

假设我们有成对的训练数据集(x̃, y)，其中x̃ = x + n，n是随机噪声。在传统的DAE中，单个cPSDAE的输入是x̃，试图最小化重建误差，即ℓ₂损失。此外，我们遵循[19, 7]的想法，引入判别器和比较器来帮助训练cPSDAE，发现它可以产生比仅优化G更清晰或更高质量的图像。

对于对抗训练，判别器D_ψ是一个可训练的卷积网络。我们可以保持传统GAN中的标准判别器损失，并将GAN的生成器损失添加到先前定义的DAE损失和比较器损失中。

## 4 实验

在本节中，我们提供了所提出的Inf-ADMM-ADNN的实验结果和分析，并与使用内循环求解逆问题的传统ADMM进行了比较。

### 4.1 合成数据

为了评估所提出的Inf-ADMM-ADNN的性能，我们首先在合成数据上测试神经网络K_φ，近似矩阵求逆。假设真实值**x**从拉普拉斯分布Laplace(μ, b)中抽取，其中μ = 0是位置参数，b是尺度参数。前向模型A是一个稀疏矩阵，表示步长为4的卷积。噪声**n**从标准高斯分布N(0, σ²)中抽取。因此，观测数据生成为**y** = A**x** + **n**。

遵循贝叶斯定理，给定**y**的**x**的最大后验估计可以等价地表述为优化问题。我们比较了基于梯度下降的更新、矩阵求逆的闭式解和所提出的使用预训练神经网络的无内循环更新。目标函数关于迭代次数和时间的演化已在图2的左侧和中间绘制。虽然所有三种方法在迭代方面表现相似，但所提出的无内循环方法和闭式求逆方法的收敛速度比基于梯度的方法快得多。

### 4.2 图像超分辨率和运动去模糊

我们将所提出的Inf-ADMM-ADNN应用于求解流行的图像超分辨率问题。我们在MNIST数据集和Caltech-UCSD Birds-200-2011（CUB-200-2011）数据集的11K图像上测试了我们的算法。高分辨率图像使用3×3的高斯核模糊，并在垂直和水平方向上每4个像素下采样，以生成相应的低分辨率图像。双三次插值和使用所提出的Inf-ADMM-ADNN的结果在视觉上显示出更好的结果，恢复了更多细节，包括颜色和边缘。

### 4.3 联合超分辨率和着色

虽然图像超分辨率试图从空间退化的图像增强空间分辨率，但在光谱域中存在相关应用，即从光谱退化的图像增强光谱分辨率。一个有趣的例子是自动着色，即幻觉出无色照片的合理颜色版本。据作者所知，这是第一次可以从单波段图像同时增强光谱和空间分辨率。

## 5 结论

在本文中，我们提出了一种加速的交替方向乘子法，即Inf-ADMM-ADNN，通过使用两个预训练的深度神经网络来求解逆问题。每次ADMM更新包括通过这两个网络的前向传播，复杂度与数据维度成线性阶。更具体地说，学习了一个条件像素重排去噪自编码器来执行近端算子的摊销推断。这个自编码器导致从训练数据中学习的隐式先验。一个数据无关的结构化卷积神经网络从噪声中学习，以显式求逆与前向模型相关的大矩阵，摆脱了ADMM更新中的任何内循环，与传统的基于梯度的方法形成对比。在合成和真实数据集上的实验和分析证明了所提方法的效率和准确性。在未来的工作中，我们希望将所提方法扩展到与非线性前向模型相关的逆问题。

这篇论文是本课题的**核心理论基础**：

| 论文方法 | 本课题应用 |
|----------|-----------|
| H估计网络（无数据训练） | 用于学习退化算子的逆变换 |

**关键创新点**：
1. H估计网络**不需要真实水下图像数据**，用随机噪声训练
2. 矩阵求逆用神经网络学习，避免迭代计算
3. ADMM每次迭代复杂度与数据维度成线性关系
